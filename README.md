# Munna Catering Frontend

A client-facing website for Munna Catering Services, built to present offerings clearly and drive real-world inquiries. The project is a traditional multi-page frontend with a small build layer for JavaScript output, keeping the delivery straightforward and hosting-friendly.

## Purpose

This repo exists to showcase services, improve trust, and present the catering business online in a polished way without overengineering the stack.

## Stack

- HTML
- CSS
- Vanilla JavaScript
- Babel-based JavaScript build step

## Project structure

- `index.html` - main entry point
- `html/` - supporting internal pages
- `css/` - stylesheets
- `js/` - source JavaScript
- `dist/` - built output
- `images/` - visual assets
- `package.json` - build and lint scripts

## Local development

1. Install dependencies with `npm install`.
2. Open `index.html` for static review or serve the root with a local server.
3. Run `npm run build` when you want compiled JavaScript in `dist/`.
4. Use `npm run lint` for a quick lint pass.

## Notes

- Current local package version is `1.0.0`.
- The repo is intentionally simple and practical, which suits brochure-style business sites well.
